# Netflo - Plateforme de BAT

Plateforme moderne de gestion et de validation de BAT (Bon À Tirer) avec workflows personnalisables.

## Fonctionnalités

- Gestion de projets et de documents
- Validation avancée avec workflows personnalisables
- Annotations et commentaires en temps réel
- Notifications multicanales
- Interface intuitive et moderne

## Installation

1. Cloner le repository
2. Installer les dépendances : `npm install`
3. Configurer l'environnement
4. Lancer l'application : `npm start`

## Structure du projet

```
netflo/
├── backend/        # API Node.js
└── frontend/       # Interface React
```

## Technologies utilisées

- Frontend : React, TailwindCSS
- Backend : Node.js, Express
- Base de données : PostgreSQL
